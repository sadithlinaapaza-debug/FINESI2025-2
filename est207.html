Descripción del Curso

La Programación Numérica combina fundamentos de programación con métodos numéricos para resolver problemas matemáticos complejos. Este curso introduce algoritmos computacionales, métodos de aproximación numérica, y técnicas de optimización.

Objetivos Principales

  • Algoritmos numéricos fundamentales
  • Métodos de aproximación y error
  • Optimización computacional

Herramientas

  • Python / MATLAB
  • NumPy / SciPy
  • Visualización de algoritmos
