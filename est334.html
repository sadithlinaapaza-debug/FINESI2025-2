Descripción del Curso

La Estadística Espacial es una rama especializada que se encarga del análisis de datos que presentan dependencia espacial. Este curso aborda métodos geoestadísticos, análisis de patrones espaciales, y técnicas de modelado para datos con componente geográfico.

Objetivos Principales

  • Análisis geoestadístico avanzado
  • Modelado de autocorrelación espacial
  • Técnicas de interpolación kriging

Herramientas

  • GIS / QGIS
  • R Spatial / Python GeoPandas
  • Análisis de datos geoespaciales
